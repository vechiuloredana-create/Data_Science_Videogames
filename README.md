# Data_Science_Videogames
This project explores a video game sales dataset to answer several questions about the industry:

Which publisher has the highest sales per game? <br>
What platforms have the highest global sales? <br>
Which decade produced the best-selling games?<br>
What features correlate with high user scores?<br>
How have video games changed over the years?<br>
How does genre popularity compare across regions?<br>


Which publisher has the highest sales per game?<br>
On average Palcome has the highesr sales per games followed by Red Orb.

What platforms have the highest global sales?<br>
The platform with the highest global sales is PS2, while the lowest is PCFX.
